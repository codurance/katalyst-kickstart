# Codurance :: Katalyst Kickstart

## Node Project

There are three projects, FizzBuzz, LeapYear and RomanNumerals implemented with javascript and use Jest to test it.

## Getting Started 

First all, you need Node install on your computer, so just run `cd FizzBuzz`, install dependence using `npm install` 


## Start the test

To start the test in your repo, you need to run `jest` in your terminal to see the results.
